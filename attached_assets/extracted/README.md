# Quest Island — Loop 1 (Replit-ready)

This is a Vite + React + TypeScript project with Tailwind CSS.

## Run on Replit

1. New Repl → **Import from Zip** → upload this archive.
2. If Replit doesn't auto-install deps, open Shell and run: `npm install`.
3. Click **Run** (or run `npm run dev`).

## Swap in your app

- Open `src/App.tsx` and paste your canvas code (it must `export default` a React component).
- Ensure in `index.html` we load `/src/main.tsx` and Tailwind is present via `src/index.css`.

## Useful files
- `vite.config.ts` — Vite with `@vitejs/plugin-react`
- `tailwind.config.cjs` + `postcss.config.cjs`
- `tsconfig.json` — TypeScript config
