LearnOz Teacher Quick Start Guide

Get started in 3 simple steps

1

Set Up Your Class

Create your class: Go to Teacher Panel → Classes → "Add New Class"

Get your class code: Share the 6-digit code with students for easy joining

Customize settings: Set grade level, learning goals, and assessment preferences

2

Add Students

Bulk import: Upload a CSV file with student names and details

Manual entry: Add students one by one through the Students tab

Self-registration: Students can join using your class code

3

Assign Learning Path

Choose content: Select from Australian Curriculum-aligned lessons

Set parameters: Choose difficulty level and learning objectives

Monitor progress: Track student advancement through the dashboard

💡 Pro Tips for Success

Projector Mode: Press ? then enable projector-safe mode to hide student names and increase font size for classroom presentations.

Offline Learning: Content auto-caches for uninterrupted learning even with poor connectivity.

Progress Monitor: Use the Timeline view to see real-time student activity and identify who needs help.
