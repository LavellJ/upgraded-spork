# LearnOz Teacher Quick Start Guide

## 🚀 Get Your Classroom Learning in 3 Steps

---

### Step 1: Set Up Your Class
**⏱️ Time: 2 minutes**

1. **Access the Guide**
   - Click the gear icon (⚙️) in the top navigation
   - Toggle "Guide Mode" to enable teacher features

2. **Create Your Class**
   - Navigate to Guide → Classes
   - Click "Add New Class"
   - Enter your class name (e.g., "5B Mathematics")
   - Your unique class code will be generated automatically

*🖼️ [Screenshot: Class creation interface with code generation]*

**✅ Success Check:** You can see your class code displayed clearly

---

### Step 2: Add Students to Your Class
**⏱️ Time: 5 minutes**

**Option A: Individual Setup**
- Share your class code with students
- Students visit the LearnOz URL and enter the code
- They create their own learner profiles

**Option B: Bulk Import (Recommended)**
- Go to Guide → Classes → Import Students
- Upload a CSV file with student names
- System generates individual learner profiles
- Print login credentials for distribution

*🖼️ [Screenshot: CSV import interface and format example]*

**✅ Success Check:** Student names appear in your class roster

---

### Step 3: Assign Your First Learning Path
**⏱️ Time: 3 minutes**

1. **Choose Content**
   - Navigate to Guide → Assignments
   - Click "Create New Assignment"
   - Select from Australian Curriculum packs:
     - Mathematics (Years 3-6)
     - Science (Years 3-6)
     - Literacy (Years 3-6)

2. **Set Parameters**
   - Choose lessons for the week
   - Set due dates (optional)
   - Add assignment notes for students

3. **Assign to Class**
   - Select your class from the dropdown
   - Click "Assign to Class"
   - Students see the assignment immediately

*🖼️ [Screenshot: Assignment creation flow with curriculum selection]*

**✅ Success Check:** Assignment appears in student dashboards

## 🎯 Pro Tips for Success

### Classroom Display Settings
- **Projector Mode**: Guide → Settings → Display → Projector Safe
- **Font Size**: Increase to 120% for better visibility
- **High Contrast**: Enable for students with visual needs

### Offline Learning
- **Download Content**: Assignments automatically cache for offline use
- **Sync Status**: Green dot = online, Orange = syncing, Red = offline
- **Best Practice**: Let devices sync for 5 minutes before going offline

### Progress Monitoring
- **Real-time Dashboard**: Guide → Timeline shows live student activity
- **Weekly Reports**: Auto-generated every Sunday
- **Parent Summaries**: Export from Guide → Reports
