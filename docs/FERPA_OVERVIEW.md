# FERPA Overview - LearnOz

*Family Educational Rights and Privacy Act compliance for classroom deployments*

## Our Role as School Official

When LearnOz is used in educational settings, we operate under the **School Official Exception** (§99.31(a)(1)) with **legitimate educational interests**.

### School Official Status
- **Direct Educational Service**: Providing personalized learning analytics and curriculum tools
- **Under Direct Control**: Operating under school's data governance policies  
- **Legitimate Interest**: Supporting instruction, assessment, and educational planning
- **Same Controls**: Subject to same privacy restrictions as school employees

## Educational Records Covered

LearnOz may handle the following FERPA-protected educational records:

### Student Progress Data
- Learning activity completion and performance
- Time-on-task and engagement metrics
- Curriculum alignment and standards mastery
- Assignment completion and assessment results

### Directory Information
**Default Position**: We do **NOT** treat any student information as directory information that can be disclosed without consent.

**School Discretion**: Schools may designate certain elements (like student names within classes) as directory information per their own policies.

## Parent and Student Rights

### Access Rights (§99.10)
**Under 18**: Parents have the right to:
- Inspect and review their child's educational records in LearnOz
- Request copies of progress reports and learning analytics
- Receive explanations of data collected and how it's used

**18+ Students**: Students have direct rights to their own educational records

### Amendment Rights (§99.20)
- Parents/students may request correction of inaccurate records
- Formal hearing process available if school denies amendment request
- Right to place explanatory statement in record if amendment denied

### Disclosure Control (§99.30)
- No disclosure without written consent except for specific FERPA exceptions
- School officials with legitimate educational interests (including LearnOz)
- Authorized representatives for audit/evaluation purposes
- Directory information (only if school has proper policies and notices)

## Data Handling Practices

### Legitimate Educational Interests
LearnOz accesses student data only for:
- **Instructional Support**: Personalizing learning experiences and recommendations
- **Progress Monitoring**: Tracking academic growth and identifying learning needs
- **Curriculum Planning**: Helping teachers adjust instruction based on learning analytics
- **Assessment**: Supporting formative and summative evaluation processes

### Prohibited Uses
We do **NOT** use educational records for:
- Commercial marketing to students or families
- Advertising or promotional activities  
- Non-educational research without proper consent
- Sharing with unauthorized third parties

## Data Subject Requests

### Export Requests
- Parents/students can request complete export of all educational records
- Data provided in machine-readable format (JSON/CSV)
- Includes learning progress, time logs, and system interactions
- Delivered within 45 days per FERPA requirements

### Deletion Requests  
- Parents/students can request deletion of educational records
- School retains final authority over educational record retention
- Personal identifiers removed while preserving aggregate analytics for school use
- Complete deletion available upon student withdrawal or program termination

### Correction Requests
- Technical errors in data collection corrected immediately
- Disputes about learning progress require educator review
- Parents/students may add explanatory statements to contested records

## School Responsibilities

### Annual Notification
Schools must include LearnOz in their annual FERPA rights notification:
- Notice that LearnOz operates as a school official
- Description of legitimate educational interests
- Contact information for data access requests

### Directory Information
If schools designate student names/classes as directory information:
- Must provide annual opt-out opportunity to parents
- Must honor parent opt-out requests in LearnOz system
- Cannot disclose opted-out students in class rosters or reports

### Training and Oversight
- School staff using LearnOz must understand FERPA obligations
- Regular review of data access logs and usage patterns
- Incident response procedures for any unauthorized disclosures

## Technical Safeguards

### Access Controls
- Role-based permissions ensuring staff access only appropriate student records
- Authentication required for all educational record access
- Session timeouts and automatic logouts for security

### Audit Logging
- Complete logs of who accessed which student records when
- Automated alerts for unusual access patterns
- Quarterly access reports provided to school administrators

### Data Security
- Encryption in transit and at rest for all educational records
- Regular security assessments and vulnerability testing
- Incident response plan for any potential data breaches

## Compliance Monitoring

### Regular Reviews
- Annual assessment of school official designation appropriateness
- Quarterly review of data access patterns and usage
- Continuous monitoring of parent/student rights fulfillment

### Documentation
- Maintenance of all consent forms and opt-out requests
- Records of parent/student access requests and responses
- Documentation of any educational record amendments or corrections
